# starred2csv
 
Export your starred repositories from GitHub to a CSV file.

### Installation

```bash
go get -u github.com/TBXark/starred2csv
```

### Usage

```bash
starred2csv -username=TBXark -token=YOUR_GITHUB_TOKEN
```
